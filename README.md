# Tic Tac Toe

Ein interaktives Tic Tac Toe Spiel mit Einzelspieler- und Mehrspieler-Modus, entwickelt mit HTML, CSS und JavaScript.

## Features

- **2-Spieler-Modus**: Spiele lokal gegen einen Freund
- **1-Spieler-Modus**: Spiele gegen den Computer mit drei Schwierigkeitsgraden
- **Intelligente KI**: Minimax-Algorithmus für optimales Spiel
- **Modernes Design**: Responsive UI mit Animationen und Farbverläufen
- **Gewinnhervorhebung**: Visuelles Feedback für Gewinnkombinationen

## Schwierigkeitsgrade

### Leicht
Der Computer macht zufällige Züge.

### Mittel
Der Computer mischt optimale und zufällige Züge (50/50).

### Schwer
Der Computer nutzt den Minimax-Algorithmus und spielt optimal. Praktisch unschlagbar!

## Installation

Keine Installation erforderlich! Einfach die Datei herunterladen und im Browser öffnen:

1. Lade die `tic-tac-toe.html` herunter
2. Öffne die Datei in deinem Browser
3. Spiel starten!

## Verwendung

1. Wähle den Spielmodus (2 Spieler oder Gegen Computer)
2. Bei 1-Spieler: Wähle den Schwierigkeitsgrad
3. Klicke auf ein Feld, um deinen Zug zu machen
4. Versuche drei in einer Reihe zu bekommen (horizontal, vertikal oder diagonal)
5. Klicke auf "Neues Spiel" um von vorne zu beginnen

## Technische Details

- **Keine Abhängigkeiten**: Alles in einer einzigen HTML-Datei
- **Vanilla JavaScript**: Kein Framework erforderlich
- **Responsive Design**: Funktioniert auf Desktop und Mobile
- **Minimax-Algorithmus**: Implementiert für die KI im schweren Modus
